Conceitos de HTML e CSS Utilizados

Para o desenvolvimento do layout, utilizamos o display: flex, que foi muito útil para poder alinhar itens em forma de coluna ou em linha, e depois alinhá-los em relação à div.
Para a criação do menu, utilizamos alguns conceitos básicos de transições, que também foram reutilizados na página "grupo", onde é possível notar o hover ocultando um pouco a imagem e fazendo aparecer um texto.
Utilizamos, também, a criação de classes contendo "width" padrão, facilitando o código CSS para que não houvessem muitas repetições de proprieades.
E, por fim, usamos o CSS com "background-image" para controlar o plano de fundo, e manter "img" somente em casos de imagens que realmente seriam aplicadas em alguma div ou conteúdo específico.
